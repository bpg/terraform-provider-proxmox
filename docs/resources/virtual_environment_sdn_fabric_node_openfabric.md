# Resource: proxmox_virtual_environment_sdn_fabric_node_openfabric

OpenFabric Fabric Node in Proxmox SDN. Fabrics in Proxmox VE SDN provide automated routing between nodes in a cluster.

## Example Usage

```terraform
resource "proxmox_virtual_environment_sdn_fabric_openfabric" "main" {
  id        = "main-fabric"
  ip_prefix = "10.0.0.0/16"
}
resource "proxmox_virtual_environment_sdn_fabric_node_openfabric" "main" {
  fabric_id       = proxmox_virtual_environment_sdn_fabric_openfabric.main.id
  node_id         = "pve"
  ip              = "10.0.0.1"
  interface_names = ["nic0", "nic1"]
}
```
