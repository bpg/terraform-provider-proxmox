# Resource: proxmox_virtual_environment_sdn_fabric_node_ospf

OSPF Fabric Node in Proxmox SDN. Fabrics in Proxmox VE SDN provide automated routing between nodes in a cluster.

## Example Usage

```terraform
resource "proxmox_virtual_environment_sdn_fabric_ospf" "main" {
  id        = "main-fabric"
  ip_prefix = "10.0.0.0/16"
  area      = "0"
}
resource "proxmox_virtual_environment_sdn_fabric_node_ospf" "main" {
  fabric_id       = proxmox_virtual_environment_sdn_fabric_ospf.main.id
  node_id         = "pve"
  ip              = "10.0.0.1"
  interface_names = ["nic0", "nic1"]
}
```
